A sample project to show how to use Next.js with 
- SSR, 
- dynamic Routing, 
- Redux store and 
- Using Identity Providers
- TypeScript
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
